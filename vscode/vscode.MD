#### Debug pytest
`.vscode/launch.json`:
```
{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Run pytest",
            "type": "debugpy",
            "request": "launch",
            "module": "pytest",
            "console": "integratedTerminal",
            "args": ["-v", "-s"],
            "cwd": "${workspaceFolder}"
        }
    ]
}
```
