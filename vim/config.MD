Config file: `~/.vimrc`  
Edit config inside vim: `:spit ~/.vimrc`

Example vim config:
```
" Basic
set cursorline " highlight cursor line 
```

### Dunamically add settings
:set `<setting>` | Effect
-|-
ff=unix | dos2unix
nu | line num
nu rnu | line num + relative numbering
ic | search case insensitive
